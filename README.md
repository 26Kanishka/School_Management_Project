# School_Management_Project

This repository contains a comprehensive School Management System developed using Python and SQL. The system provides a user-friendly interface for managing school's operations.

Features

1. Student Record Management:

* Add new student records with details including name, stream, roll number, date of birth, gender, and email.
* Update existing student records to keep information current.
* Delete student records when necessary.

2. Data Retrieval:

* Search for student records based on various criteria such as name, roll number, or stream.
* View all student records in a user-friendly format.

Technologies Used

Python: The system is built using Python, a versatile and powerful programming language.

SQLite: The project uses SQLite, a lightweight and serverless SQL database engine, for data storage.
